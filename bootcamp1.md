project

assets/

```javascript

 return(
  <View>

  </View>
 )
```

# 2. Ejemplo de app tipo Gymkana universitaria

Características:

- React Native
- Geolocalización
- Pistas en diferentes puntos
- Ranking de jugadores

Tecnologías:

Frontend
React Native + Expo

Backend
Node.js o NestJS

Base de datos
Firebase o MongoDB

Funcionalidades:

- Login de usuario
- Mostrar mapa
- Detectar posición GPS
- Activar preguntas al llegar a un punto
